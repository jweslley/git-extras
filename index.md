
# Git Extras

 Little git extras.

## Installation

Clone / Tarball:

     $ make install

One-liner:

    $ curl http://github.com/visionmedia/git-extras/raw/master/bin/git-update-extras | sh

Brew (buggy):

     $ brew install git-extras
